## 编译环境
1、从serverbase/ 项目中引入 lib/ bin/ include/ 目录

    ```
    ln -s -f /data/work_git/serverbase/lib lib
    ln -s -f /data/work_git/serverbase/include include
    ln -s -f /data/work_git/serverbase/bin bin
    ```

2、```mkdir debug```
